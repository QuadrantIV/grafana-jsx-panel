# Grafana Jsx Panel
一个 jsx 实时渲染的 grafana 面板插件

简体中文 | [English](./README.md)

## Feature
- 🔄 基于 @babel/standalone 实时转码，可通过 jsx 语法代码来自定义组件面板
- ✏️ 良好的代码编辑体验
- 📚 可直接使用 @grafana/ui 组件库
